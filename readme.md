# Inventory Controll

A inventory controller app with NFC support

## NFC
- Tag Id

## Enitities

### tag
	- id 			=> 0xBB3B7340
	- name 			=> Pneu Good Year 205/55
	- description 	=> Pneu para carro

### inventory
	- @Item::id		=> 0xBB3B7340
	- quantity 		=> 3
	- validity		=> 29/04/2023
